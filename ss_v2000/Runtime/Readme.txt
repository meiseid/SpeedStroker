>>>>あそびかた概要>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

１、このフォルダにある「Sstroker.exe」が実行ファイルです。
　　ダブルクリックして起動します。

２、タイトル画面が表示されます。「GAME START」にカーソルを
　　あわせ、クリックまたはリターンキーを押してください。

３、ジャンル選択画面が表示されます。お好みのジャンルにカーソル
　　をあわせ、クリックまたはリターンキーで確定してください。

４、ゲームが開始します。表示されている問題文を見て、その下に
　　あるローマ字読みをひたすら打ち込んでください。入力は
　　小文字大文字のどちらでもOKです。

５、設定した問題数をクリア、またはタイムオーバー＆ミスタイプが
　　難易度に応じた許容範囲を超えるとゲーム終了となります。

６、あなたの成績が画面に表示されます。得点がハイスコアに
　　ランクインした場合、名前入力を求められます。この場合、
　　３文字までのアルファベットを入力して下さい。
　　リターンキーを押すか、「戻る」でジャンル選択画面に戻ります。

------------------------------------------------------------------
>>>>注意点>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

１、本ソフトではローマ字入力のみが使用できます。日本語入力は
　　できません。MS-IME、ATOKなどのFEPはオフにして御使用下さい。

------------------------------------------------------------------
>>>>SpeedStrokerの機能>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

１、どの場面でも、エスケープ(ESC)キーを押せばSpeedStrokerを終了
　　させる事ができます。

２、SpeedStrokerは入力に強烈な柔軟性を持っています。例えば

　　　　zya zi zyu zye zyo は ja ji ju je jo でもOK
　　　　sya si syu sye syo は sha shi shu she sho でもOK
　　　　tya ti tyu tye tyo は cha chi chu che cho でもOK
　　　　xa xi xu xe xo は la li lu le lo でもOK
　　　　ka ku ko は ca cu co でもOK
　　　　si se は ci ce でもOK
　　　　tu は tsu、hu は fuでもOK

　　といった具合です。

------------------------------------------------------------------
>>>>各画面の詳しい説明>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
------------------------------------------------------------------
１、タイトル画面 -- 各項目にカーソルを合わせ、クリックまたは
　　　　　　　　    リターンキーを押して下さい。

　　GAME START       : ジャンル選択画面に移ります。
　　GAME OPTION      : 設定変更画面に移ります。
　　HIGH SCORES      : ランキング画面に移ります。
　　EXIT APPLICATION : SpeedStrokerを終了します。
------------------------------------------------------------------
２、設定変更画面 -- ゲームに関わる様々な設定を変更できます。
　　　　　　　　    リターンキーまたは「設定終了」で、タイトル
　　　　　　　　    画面に戻ります。

　　出題数 : １ゲームあたりの問題ノルマ数を設定できます。各桁の
　　　　　　 ▲or▼をクリックし、値を上下させて下さい。

　　難易度 : ゲームレベルを８段階で指定できます。希望の段階の
　　　　　　 ★をクリックし、難易度を決めて下さい。この難易度は
　　　　　　 ゲーム中のタイム制限とタイプミス制限はもちろん、
　　　　　　 スコア計算にも大きく影響します。

　　音声　 : ゲームサウンドを鳴らす場合はONを、消す場合はOFFを
　　　　　　 クリックしてください。

ハイスコア : ランキング画面に表示されるハイスコアデータの扱いです。
　　　　　　「保存」-- アプリケーションを終了してもデータは失われ
　　　　　　　　　　　ません。ディスクに保存されます。
　　　　　　「破棄」-- アプリケーション終了時にデータは初期化され
　　　　　　　　　　　ます。
------------------------------------------------------------------
３、ランキング画面 -- ハイスコアのトップ10を見ることができます。
　　　　　　　　　    「戻る」でタイトル画面に戻ります。

　　ゲームで叩き出したハイスコアはここに登録されます。
　　列にカーソルをあわせ、クリックorリターンキーを押すと、その人の
　　詳細データを見ることができます。"Difficulty Level"とは設定変更
　　画面で設定した難易度、"Total Of Strings"とは設定変更画面で設定
　　した出題数のことです。クリックorリターンキーでランキング画面に
　　戻ります。
------------------------------------------------------------------
４、ジャンル選択画面 -- ゲーム前のジャンルセレクトです。お好みの
　　　　　　　　　　    ジャンルにカーソルを合わせ、クリックまたは
　　　　　　　　　　    リターンキーで確定してください。
　　特殊項目--
　　　　　　　TITLE : タイトル画面に戻ります。
　　　　　　　NEXT  : ジャンルが１画面に入りきらない場合現れます。
　　　　　　　        クリックすると次のページを表示します。
　　　　　　　PREV  : 前のページを表示します。
------------------------------------------------------------------
５、タイピング画面 -- ただひたすらに打ち込みましょう!

　　※この画面でのエスケープ(ESC)キーだけは特別で、ゲームの
　　　ギブアップを意味します。即座に結果画面に移行します。
　　
　　※左上の"Progress"は進捗状況です。例えば 24/100 は１００問中の
　　　２４問目であることを意味しています。
------------------------------------------------------------------
６、結果画面 -- ゲーム結果を表示します。リターンキーまたは「戻る」
　　　　　　    で、ジャンル選択画面に戻ります。

　＊名前入力    あなたの得点がハイスコアのトップ10にランクイン
　　　　　　    すると、３文字のアルファベット入力を求められます。
　　　　　　    入力にはスペースキー(空白)も使えます。
　　　　　　    ←,→,BackSpaseキーなどで入力位置を動かせます。

　　ランクインすれば、結果画面を終了しても、後でランキング画面で
　　自分の成績を確認することができます。
------------------------------------------------------------------

>>>>データのカスタマイズ>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

　　SpeedStroker最大の特徴は自由にデータを変更できることです。
